Class ObjectStorage_TokenStore_Memory

(line 12)

Description




	Located in File: /lib/ObjectStorage/TokenStore/Memory.php





A temporary token store that uses memory as token storage.


This is NOT a persistent data store nor it's anywhere close to memcache.  A token stored in this object will be removed at the end of script execution.
